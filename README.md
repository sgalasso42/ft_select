# ft_select
The recoding of the function select, an initiation to termcaps library and signal catching
## Built with
C language
## Installing
- Git clone the repository
- Compile it with `make`
## Make it work
- Run `./ft_select [args...]`
## Use it
- Spacebar : select
- Return : return selected
- Delete : delete
- Backspace : delete
- arrows : moving
